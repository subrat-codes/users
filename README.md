# Users Microservice

This microservice is used to manage user details for the application.

## Technologies Used

*   Java 21
*   Spring Boot
*   Spring WebFlux (Reactive APIs)
*   H2 Database

## How to Build

To build the project, run the following command:

```bash
mvn clean package
```

## How to Run

After building the project, you can run the application using the following command. Make sure to replace `<jar-file-name>.jar` with the actual name of the generated JAR file.

```bash
java -jar target/<jar-file-name>.jar
```
